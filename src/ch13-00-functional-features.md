# 函数式语言功能：迭代器和闭包

Rust 的设计从许多现有语言和技术中汲取了灵感，其中一个重要影响是 _函数式编程_。以函数式风格编程通常包括通过将函数作为值传递、从其他函数返回它们、将它们分配给变量以供稍后执行等方式使用函数。

在本章中，我们不会讨论函数式编程是什么或不是什么的问题，而是讨论 Rust 的一些功能，这些功能类似于许多通常被称为函数式的语言中的功能。

更具体地说，我们将涵盖：

- _闭包_，一种可以存储在变量中的类似函数的构造
- _迭代器_，一种处理一系列元素的方法
- 如何使用闭包和迭代器改进第 12 章中的 I/O 项目
- 闭包和迭代器的性能（剧透警告：它们比你想象的更快！）

我们已经涵盖了一些其他 Rust 功能，例如模式匹配和枚举，它们也受到函数式风格的影响。因为掌握闭包和迭代器是编写快速、符合习惯的 Rust 代码的重要组成部分，我们将把整章都用于它们。
