# 前言

Rust 编程语言在短短几年内已经走过了很长的路，从一个由小型新兴爱好者社区创建和孵化的语言，发展成为世界上最受喜爱和需求最高的编程语言之一。回顾过去，Rust 的力量和承诺必然会吸引人们的注意并在系统编程领域占据一席之地。但并非必然的是，全球兴趣和创新的增长渗透到开源社区，并催化了跨行业的广泛采用。

在这一点上，很容易指出 Rust 提供的出色功能来解释这种兴趣和采用的爆炸式增长。谁不想要内存安全，*以及*快速性能，*以及*友好的编译器，*以及*出色的工具，以及其他一系列出色的功能？您今天看到的 Rust 语言将多年来在系统编程方面的研究与一个充满活力和热情的社区的实用智慧相结合。这门语言是为了目的而设计的，并经过精心制作，为开发人员提供了一个工具，使编写安全、快速和可靠的代码变得更容易。

但真正使 Rust 特别的是其根源在于赋予您（用户）实现目标的能力。这是一门希望您成功的语言，赋权的原则贯穿于构建、维护和倡导这门语言的核心社区。自这本权威文本的上一版以来，Rust 进一步发展成为一种真正全球化和值得信赖的语言。Rust 项目现在得到了 Rust 基金会的强大支持，该基金会还投资于关键举措，以确保 Rust 安全、稳定和可持续。

这个版本的 *Rust 程序设计语言* 是一个全面的更新，反映了语言多年来的演变，并提供了有价值的新信息。但它不仅仅是一本语法和库指南——它是对加入一个重视质量、性能和深思熟虑设计的社区的邀请。无论您是想要首次探索 Rust 的经验丰富的开发人员，还是希望完善技能的资深 Rustacean，这个版本都为每个人提供了一些东西。

Rust 的旅程一直是协作、学习和迭代的过程。语言及其生态系统的增长直接反映了其背后充满活力、多样化的社区。从核心语言设计者到临时贡献者，数千名开发人员的贡献使 Rust 成为如此独特和强大的工具。通过拿起这本书，您不仅仅是在学习一门新的编程语言——您正在加入一个使软件更好、更安全、更令人愉悦的运动。

欢迎加入 Rust 社区！

- Bec Rumbul，Rust 基金会执行董事
