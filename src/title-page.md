# Rust 程序设计语言

_作者：Steve Klabnik、Carol Nichols 和 Chris Krycho，以及 Rust 社区的贡献者_

本版本假定您使用的是 Rust 1.90.0（2025-09-18 发布）或更高版本，并在所有项目的 *Cargo.toml* 文件中设置 `edition = "2024"` 以配置它们使用 Rust 2024 版本的惯用法。关于安装或更新 Rust 的说明，请参见[第 1 章的"安装"部分][install]<!-- ignore -->，关于版本的信息，请参见[附录 E][appendix-e]<!-- ignore -->。

HTML 格式可在 [https://doc.rust-lang.org/stable/book/](https://doc.rust-lang.org/stable/book/) 在线获取，也可以通过使用 `rustup` 安装的 Rust 离线获取；运行 `rustup doc --book` 打开。

一些社区[翻译版本][translations]也可用。

本文档可在 [No Starch Press][nsprust] 以平装本和电子书格式获取。
