# 更多关于 Cargo 和 Crates.io

到目前为止，我们只使用了 Cargo 的最基本功能来构建、运行和测试我们的代码，但它可以做更多事情。在本章中，我们将讨论它的一些其他、更高级的功能，向你展示如何执行以下操作：

- 通过发布配置文件自定义构建。
- 在 [crates.io](https://crates.io/)<!-- ignore --> 上发布库。
- 使用工作空间组织大型项目。
- 从 [crates.io](https://crates.io/)<!-- ignore --> 安装二进制文件。
- 使用自定义命令扩展 Cargo。

Cargo 可以做的比我们在本章中涵盖的功能更多，所以要全面了解其所有功能，请参阅[其文档](https://doc.rust-lang.org/cargo/)。
